# <a name="slider"></a>滑桿

![滑桿](images/UX/UX_Hero_Slider.jpg)

滑杆是 UI 元件，可讓您在播放軌上移動滑杆來持續變更值。縮小滑杆可以直接或距離直接抓取滑杆來移動。 滑杆可以處理各種輸入型別，像是注視和空中碰，還有運動控制器。
