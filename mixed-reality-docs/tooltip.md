# <a name="tooltip"></a>工具提示

![工具提示](images/UX/UX_Hero_Tooltip.jpg)

工具提示是連結至另一個控制項或物件的簡短描述。 它們可用來在仔細檢查時傳達提示或額外資訊。 工具提示可協助使用者瞭解實體環境中不會直接描述的不熟悉物件。
