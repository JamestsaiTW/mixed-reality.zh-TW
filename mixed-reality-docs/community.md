## <a name="microsoft-mixed-reality-mvps"></a>Microsoft Mixed Reality MVP

Microsoft 最有價值專家（或 Mvp）是努力與社區分享其知識的技術專家。 它們一律會處於「不規則的邊緣」，而且 unstoppable 強烈鼓勵他們親自體驗新的令人興奮的技術。 他們對 Microsoft 產品和服務有非常深入的瞭解，同時也能夠整合各種平臺、產品和解決方案，以解決現實世界的問題。 Mvp 為超過4000的技術專家和跨90個國家/地區的社區領導人組成全球化團體，並由其熱情、社區精神和知識的主管來推動。 除了其卓越的技術能力之外，Mvp 一律願意協助其他人-這就是設定它們的部分。
