## <a name="spatial-coordinate-systems"></a><span data-ttu-id="56d1a-188">空間座標系統</span><span class="sxs-lookup"><span data-stu-id="56d1a-188">Spatial coordinate systems</span></span>

<span data-ttu-id="56d1a-189">所有的 3D 圖形應用程式使用[笛卡兒座標系統](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems)理解的位置和方向呈現虛擬世界中的物件。</span><span class="sxs-lookup"><span data-stu-id="56d1a-189">All 3D graphics applications use [Cartesian coordinate systems](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems) to reason about the positions and orientations of objects in the virtual worlds they render.</span></span> <span data-ttu-id="56d1a-190">這類的座標系統建立 3 個垂直軸，沿著其位置的物件： X、 Y 和 Z 軸。</span><span class="sxs-lookup"><span data-stu-id="56d1a-190">Such coordinate systems establish 3 perpendicular axes along which to position objects: an X, Y, and Z axis.</span></span>
