# <a name="mixed-reality-documentation"></a>混合實境文件

混合實境將現實世界和虛擬內容融合到混合環境中，讓實體和數位物件可以共存並加以互動。 學習為 Microsoft HoloLens 和 Windows Mixed Reality 沉浸式頭戴裝置 (VR) 建置混合實境體驗。

<h2>HoloLens 2 簡介</h2>
