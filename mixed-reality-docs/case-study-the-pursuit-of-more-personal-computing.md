# <a name="case-study---the-pursuit-of-more-personal-computing"></a>個案研究-追求更個人運算的能力

現今的機會會藉由建立產品而發現。 這些產品提供的解決方案可讓您清楚掌握未來的需要。 在混合的事實中, 這特別適用:有意義的深入解析是從實際工作開始實際操作, 包括真實的裝置、真實的客戶和真正的問題。

在 Microsoft, 我是設計團隊的一部分, 協助企業合作夥伴使用 Windows Mixed Reality 為其企業打造經驗。 過去一年來, 我們的團隊致力於 HoloLens, 並瞭解 Microsoft 的旗艦全像裝置如何為現今客戶提供價值。 與這些公司的設計人員和開發人員密切合作, 我們的團隊著重在發現在技術上不可行、不實際或不可能有 HoloLens 的解決方案。

![案例西方保留大學](images/case-western-reserve-university.jpg)<br>
*HoloAnatomy from a Case 西部預約大學*

建立這些解決方案可協助 Microsoft 的內部團隊準備新一代的運算。 瞭解個人和企業如何與混合現實、語音和 AI 等核心技術互動, 協助 Microsoft 為開發人員打造更好的裝置、平臺和工具。 如果您是設計師或探索此空間的開發人員, 請瞭解我們目前的小組從合作夥伴學習的內容, 對於準備未來的混合現實機會非常重要。

## <a name="microsofts-ambition-for-mixed-reality"></a>Microsoft 的混合現實野心

我們居住在兩個領域: 實體和數位。 兩者都有基本的優勢, 我們利用它來增強和擴充我們的能力。 我們可以與朋友交談, 並使用我們的實體感知來瞭解本文語言之類的東西, 或者我們可以增強我們與朋友聊天的能力。 到目前為止, 這兩個世界及其各自的優勢都已以根本上分隔。

**實體世界是其中一個原子和物理。** 我們會使用我們的感知來做出決策, 利用多年學習的行為, 在我們的環境中與物件和人員互動。 雖然這類互動很容易, 但我們還是會受到我們的實體能力和本質法則的限制。

**數位世界是其中一個位和邏輯。** 計算會立即進行, 而資訊可以輕鬆散佈。 不論資訊的速度和流動程度為何, 互動通常會限制為小型螢幕、抽象輸入和雜訊摘要。

![HoloStudio](images/holostudio-450px.jpg)<br>
*HoloStudio*

如果我們可以結合實體和數位世界的優勢該怎麼辦？ 這是混合現實範圍的經驗基石: 實體和數位共同存在並順暢互動的媒介。 結合這些世界建立了新的基礎, 以更自然的方式與技術互動, 也就是個人運算的演進。

由於開發人員已經開始探索深度和目前狀態的機會, 因此混合現實的範疇已經分裂。 透過沉浸式 (虛擬實境) 體驗將使用者帶入數位世界, 並利用全像攝影 (增強的現實) 體驗, 在實體世界中建立數位物件。 但是將實體世界對應到數位世界有哪些優點？ 當我們讓電腦看不到時, 會發生什麼事？

全息影像背後的基本攝影機視覺技術, 就像是一對電腦的眼睛, 可以看到您的環境:世界各地的物件, 會隨著您的改變。 瞭解您在實體世界中的內容。 這會導致大量的資訊, 也就是我們只開始瞭解的含意。

### <a name="culminating-core-technologies"></a>累積核心技術

運算通常是一項很重要的事。 抓取我們的裝置, 告訴他們您的身分, 以及我們想要的內容。 Contorting 我們的想法, 並對齊我們所說的內容, 以符合我們認為電腦需要聽到的內容。

混合現實的承諾 (尤其是在現實世界中, 使用全像攝影的經驗) 是要減少與技術互動的負擔。 減少認知負載, 因為使用者會流覽目前的運算所固有的抽象層。 我們要如何設計體驗, 不僅利用內容上的瞭解, 還能讓您更輕鬆地繪製深入解析並採取行動嗎？ 兩個參與的技術也會嘗試解決此問題:
* 語音和交談方面, 可以讓使用者透過更自然的方式與電腦進行通訊, 也就是透過文字來回應 bot, 或發出命令給交談式硬體。
* **AI**是一種可從日益複雜的資料集深入探索見解的電源體驗。 雖然 AI 是很大的主題, 但最近的進度已為依賴電腦視覺、更自然數位助理, 以及為使用者建議動作的裝置提供基礎。

Mixed reality 提供了將這些技術結合成單一使用者體驗的媒介。 聲音在戴全像耳機時, 會成為一種強大的自然輸入方法。 AI 是一種重要的加密方式, 可情境化連接實體和數位世界的大量資訊。 這就是為什麼 Sataya Nadella 將 HoloLens 稱為「旗艦電腦」, 它是三種核心技術的累積裝置。 一種平臺, 讓人類能夠更輕鬆地與裝置和服務的成長複雜度進行互動。

### <a name="less-interface-in-your-face"></a>臉部中的介面較少

將實體世界與數位世界連線的累積裝置, 可讓我們設計更自然的經驗, 而不需要繁瑣的抽象概念。 請考慮您所建立的體驗:移除抽象的阻礙時, 您的介面會保留多少？ 當您知道使用者及其內容時, 應用程式流程中的哪些部分會變更？ 有多少功能表和按鈕？

例如, 請思考混合現實中的共用體驗, 例如為科學家打造的 OnSight tool NASA 的 Jet 推進實驗室。 它們不會建立系統來查看 Martian 資料 (將資料抽象化到畫面或顯示器上), 而是將科學家帶入資料中, 將它們整合到 Mars 的表面上。

![NASA OnSight](images/nasa-onsight.jpg)<br>
*NASA/JPL OnSight*

科學家可以直接指向它, 而不是找出將注意力吸引 Martian 地質學的按鈕。 沒有任何功能表, 沒有畫筆, 有效使用工具的學習曲線。 藉由從實體世界運用我們的已知功能, 混合現實中更自然的互動可以規避深度的技術識字率, 甚至是最先進的產業工具。

同樣地, 語音和 AI 也可以像這樣的經驗擴充自然互動。 當數位助理可以在世界各地「觀看」時, 交談會比較不繁瑣。 NASA 工具中的 bot 可能會從科學家授予 over Martian 地質學將內容解壓縮, 這在科學家手勢並詢問「this」或「it」時, 做為資訊的準備 (和交談) 來源。 知道您的內容的電腦已準備好使用適合您的資訊, 透過最適合您的內容的互動方法來進行。

### <a name="building-on-a-foundation"></a>建立基礎

在這一世界的內容經驗中, 使用者的行動性動力就是關鍵。 使用高度個人化的內容, 在裝置與服務之間移動流暢地, 使用者將能順暢地與實體世界分享, 並建立*大規模*的平臺挑戰。 當實體世界變成數位端點時, 互通性成為頂尖優勢。


> *「結合數位世界與實體世界將會讓人的潛能發揮最大效果 。讓地球上的每個人和組織都能達到更多目標。」*<br>
> -Satya Nadella

Windows Mixed Reality 是為裝置生態系統建立平臺的一項工作, 讓開發人員針對最大的可能物件建立沉浸式、實惠且相容的體驗。 未來將不會受限於單一制造商, 而只讓單一裝置。 耳機、行動裝置、電腦、配件 。 所有這些實體專案都必須透過基礎平臺交互操作 (以及圖形和服務之類的數位專案), 才能成功地提供混合現實的承諾。

## <a name="designing-for-tomorrows-experiences-today"></a>今天為明天的經驗設計

![今天為明天的經驗設計](images/designing-for-tomorrows-experiences-today-450px.jpg) 
<br>
這一新類別體驗背後的每一項核心技術, 都讓設計人員和開發人員立即建立引人注目且成功的體驗。 藉由減少抽象概念, 我們可以直接與數位世界互動, 讓我們能夠以增強和擴展人力能力的方式進行設計。 語音技術 (透過像是 Cortana 的 bot 和數位助理) 可讓使用者執行日益複雜的交談和案例, 而 AI 技術 (透過 Microsoft 認知服務之類的工具) 會促使公司重新思考使用者如何會與社交媒體的所有專案互動, 以供應鏈管理。

這些類型的互動會依賴新類別的設計工具, 以及平臺的基本支援。 建立這些工具並為裝置和服務建立此平臺, 依賴于瞭解明天的經驗如何解決真正的有形問題。 我們已識別出五個商機領域, 其中企業合作夥伴已提供寶貴的解決方案, 而我們相信持續的投資將協助我們為這一種新的計算類別做好準備。 

## <a name="areas-of-opportunity"></a>商機的範圍

過去一年的開發人員合作關係已經發現與客戶迴響的機會, 並建立成功的企業解決方案。 從科學家和技術人員到設計人員和用戶端, 已發生五個商機領域, Microsoft 合作夥伴會發現混合現實的價值。 這些領域已提供大規模的深入資訊, 包括 Windows Mixed Reality 等平臺的未來需求, 並可協助您瞭解這些新體驗會如何影響我們學習、共同作業、溝通和建立的方式。

### <a name="1-creation-and-design"></a>1.建立和設計

混合現實的其中一個主要機會, 就是能夠以真正的規模和規模, 即時查看及操作3D 設計。 設計和原型化工具已將畫面的範圍進行轉義, 而回到設計領域, 通常會保留給有形的實體材質, 像是木材和 clay。

<br>

>[!VIDEO https://www.youtube.com/embed/Hx6biWE2VsM]

[Autodesk](https://www.youtube.com/watch?v=Hx6biWE2VsM)建立了一個混合現實的經驗, 旨在改善整個產品開發流程的共同作業。 工程師和設計人員在其環境中調查並 critique 3D 模型的功能, 可讓他們即時反復查看設計。 這不僅能夠更快速地進行原型設計, 也能在程式中提供更自信的決策。

像這樣的體驗強調了核心共同作業體驗的需求, 也就是讓使用者能夠查看和與共享物件通訊的能力。 Autodesk 的目標是要將產品從 design professional 和工程師擴充到數位演出者、學生和業餘人士。 隨著使用者的3D 專長層級降低, 與物件互動的能力自然變得很重要。

### <a name="2-assembly-and-manufacturing"></a>2.元件和製造

從工廠樓層增加的特製化, 到供應鏈管理的快速進步, 相關資訊的無縫存取就是關鍵。 Mixed reality 提供合成大量資料集的功能, 並提供視覺化顯示來協助導覽和操作等領域。 這些通常是高度技術性的小型欄位, 其中與自訂資料集和服務的整合非常重要, 可降低複雜性並提供成功的經驗。

<br>

>[!VIDEO https://www.youtube.com/embed/8OWhGiyR4Ns]

電梯製造商[ThyssenKrupp](https://www.youtube.com/watch?v=8OWhGiyR4Ns)為電梯服務技術人員建立了經驗, 讓他們能夠在準備作業時將問題視覺化並加以識別。 有了橫跨24000技術人員的團隊, 像是 HoloLens 這類裝置可讓這些技術人員有遠端、無人參與的技術和專家資訊存取權。

ThyssenKrupp 強調了一種功能強大的概念, 可讓您快速地將重要、內容相關的資訊傳遞給使用者。 當我們仔細探討新的體驗時, distilling 與使用者相關之內容的大量可能資訊將會是關鍵。

### <a name="3-training-and-development"></a>3.訓練與開發

以三個維度代表物件和資訊, 可提供新的方式來說明以視覺化方式呈現的案例, 並透過空間理解。 訓練成為一個重要的商機領域, 利用數位代表龐大、複雜的物件 (例如 jet 引擎) 來建立訓練模擬, 以提供實體解決方案成本的一小部分。

<br>

>[!VIDEO https://www.youtube.com/embed/GjZgI2oDcwM]

[日本航空公司](https://www.youtube.com/watch?v=GjZgI2oDcwM&t=158s)已經試驗過概念計畫, 為引擎機制和航班小組提供補充訓練。 大規模的 jet 引擎 (大小和複雜度) 可以3D 表示, 忽略實體世界在受訓者上移動和旋轉虛擬物件的限制, 讓他們能夠查看航空公司元件如何即時工作。

使用虛擬元件進行定型 (並減少昂貴、實體訓練模擬器的需求) 是在企業案例中提供價值的主要方式。 隨著此案例的擴充 (如同我們在醫學之類的領域所見), 電腦視覺對於辨識環境中的唯一物件、瞭解使用者的內容, 以及提供相關的指示特別重要。

### <a name="4-communication-and-understanding"></a>4.溝通與瞭解

兩個人之間的互動 (不論這兩個參與者是在混合現實裝置中, 還是在傳統電腦或電話上), 都可以在新環境中提供深度, 或是在與其他人溝通時, 有意義的存在。 在企業案例中, 這種狀態是行動小組的 boon, 可協助提高對專案的瞭解, 並減少旅行的需要。

<br>

>[!VIDEO https://www.youtube.com/embed/kXVW4sUsh3A]

商業技術製造商[Trimble](https://www.youtube.com/watch?v=kXVW4sUsh3A)開發了一種解決方案, 可供架構和結構產業專業人員在建立開發期間共同作業和審查工作。 專業人員可以從遠端 immerse 自己到專案中, 以討論進度, 或是在其周圍的環境中, 以其最終形式呈現的位置和審核計畫。

共用體驗是 Microsoft 投資的主要領域, 其中的應用程式 (例如 Skype) 會探索新的方式來表示網際空間中的人類。 小組正在探索體積型的影片錄影、虛擬人偶, 以及參與者實體空間的重建。

### <a name="5-entertainment-and-engagement"></a>5.娛樂與 engagement

混合現實的深度本質可能會對客戶與娛樂和行銷的互動方式產生巨大的影響。 演出者和娛樂工作室已探索混合現實做為故事行銷的吸引人, 而公司則是探索品牌行銷的含意。 從私用 showrooms 中的產品示範, 到在貿易 expos 中所說的願景, 內容都可以更沉浸且量身打造的方式提供。

<br>

>[!VIDEO https://www.youtube.com/embed/DilzwF90vec]

Volvo 建立了展示其最新汽車機型的體驗 (以不同的色彩和設定 immersing 使用者), 同時強調先進的感應器如何工作, 以建立更好、更安全的駕駛體驗。 讓客戶透過引導式 showroom 體驗, 可讓 Volvo 告訴您在最新汽車功能背後的故事, 同時又能提供令人印象深刻的品牌故事圖解。

娛樂有好幾種方式可以推送混合現實的界限 (尤其是虛擬實境), 而且在此空間中最吸引人的是, 它會與先前的商機領域結合: 通訊和瞭解。 有了多個使用者, 每一個都有自己的裝置和介面方法變體, 您可以想像在娛樂領域中個人化的未來。

## <a name="start-building-today"></a>立即開始建立

我們很難說, 消費者的未來混合現實會是什麼, 但專注于獨特的問題、實際的硬體實際操作, 以及 ***目前* 使用混合現實、語音和 AI 的交集進行實驗的關鍵**。 Microsoft 剛開始使用混合現實, 但是從現今企業所實現的成功學習, 將可協助您建立未來的經驗。
