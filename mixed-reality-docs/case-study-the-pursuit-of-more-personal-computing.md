# <a name="case-study---the-pursuit-of-more-personal-computing"></a>案例研究-追求更多個人運算

未來的機會是未發現立即建置產品。 這些產品提供的解決方案會顯示所前進未來。 與混合實境更是如此：取得實際操作實際工作是來自有意義的深入解析，實際的裝置、 真實的客戶，真正發生的問題。

在 Microsoft，我協助建置使用 Windows Mixed Reality 其業務的經驗的企業合作夥伴設計團隊的一部分。 過去一年來，我們的團隊的重點在於 HoloLens 和了解 Microsoft 的旗艦全像攝影版裝置如何提供值給客戶立即。 使用密切設計人員和開發人員從這些公司，我們的小組著重於技術上來說並不可行、 財務其實並不切實際，或沒有 HoloLens，否則不可能是找出解決方案。

![案例 Western 保留大學](images/case-western-reserve-university.jpg)<br>
*從案例 Western 保留 University HoloAnatomy*

建置這些方案可協助 Microsoft 的內部團隊準備新一代的運算。 學習個人和企業與混合的實境、 語音和 AI，這類的核心技術互動的方式，可協助 Microsoft 開發人員建置更好的裝置、 平台和工具。 如果您是設計工具或開發人員探索此空間中，了解我們的團隊向的學習合作夥伴立即務必做好未來的混合的實境機會。

## <a name="microsofts-ambition-for-mixed-reality"></a>Microsoft 的發展企圖的混合實境

我們在兩者間即時生活： 實體與數位。 兩者都有基本的優點，我們利用增強和擴充我們的能力。 我們可以討論親自與朋友，使用我們實體的角度看來，若要了解事情像是身體語言或我們可以增加我們對談的影片-聊天與朋友從英哩遠的能力。 現在這兩個世界，而且其各自的優點，直到已完全分隔。

**真實世界是其中一個原子和物理條件。** 我們可以使用我們的角度看來，做出決策，利用多年的學到的行為，與物件和我們的環境中的人互動。 儘管這些互動的方便，我們都受到我們實體的能力和法律的本質。

**數位世界是其中一個位元和邏輯。** 可讓您輕鬆散發資訊時，會立即進行計算。 儘管的速度和資訊流程的互動為太小螢幕，抽象的輸入，並有很多雜訊的摘要通常有限。

![HoloStudio](images/holostudio-450px.jpg)<br>
*HoloStudio*

如果我們無法將合併實體與數位世界的優點嗎？ 這是體驗的基石與多個混合實境： 在媒體中的實體和 digital 同時存在而順暢地互動。 結合這些領域建立新的基礎，讓較自然地互動技術 — 在個人電腦運算的發展。

混合實境各種具有分裂式，因為開發人員已經開始探索的深度和目前狀態的機會。 帶入沈浸式 （虛擬實境） 體驗數位世界中的使用者，並在真實世界全像攝影版 （擴增的實境） 體驗建立數位的物件。 但是，對應數位世界的真實世界的優點有哪些？ 當我們提供給電腦若要查看的眼睛，會發生什麼事？

全像投影的基本相機願景技術的作用就像一對眼睛的電腦，以查看您四周圍環境：物件，在世界中，人員解決您的變更發生時。 數位的了解您在真實世界的內容。 這會造成極大的資訊，其中的含意我們才開始了解。

### <a name="culminating-core-technologies"></a>重點的核心技術

運算往往是具體的功能。 抓取我們告訴它們我們的使命，以及我們想要的裝置。 Contorting 我們的想法，以及對齊這些假設以符合我們相信電腦必須了解。

混合實境，尤其是在全像攝影版的體驗，真實世界的承諾，是降低與科技互動的負擔。 當使用者瀏覽的立即運算固有的抽象層，請減少認知負載。 我們要如何設計體驗，不僅利用關聯式的理解，讓您更輕鬆地深入了解，並採取動作？ 兩個參與的技術也會嘗試解決這個問題：
* **語音**、 語音和交談，則可讓使用者與電腦透過更自然的方式進行通訊 — 回答 bot 透過文字或發出命令至交談式的硬體。
* **AI**精巧型去蕪存菁日益複雜的資料集的深入剖析的體驗。 AI 雖然很龐大的主題，最近進度提供基礎依賴電腦視覺、 更自然的數位助理，及建議的動作給使用者的裝置。

混合的實境提供將這些技術合併成單一的使用者體驗的媒體。 頭戴耳機全像攝影版時，語音就會變成功能強大且自然的方法輸入。 AI 扮演重要的 cipher contextualize 大量連接實體與數位世界的資訊。 這就是為什麼 Sataya Nadella 指的是 「 ultimate 的電腦 」 為 HoloLens、 三個核心技術，它會是重點的裝置。 平台能夠讓使用者更輕鬆地進行互動的裝置和服務持續增加的複雜性。

### <a name="less-interface-in-your-face"></a>較少的介面，在您的面孔

將連接數位世界的真實世界的重點裝置可讓我們設計的體驗，而不需要很麻煩的抽象概念的更自然，放入。 您已建立的體驗，請考慮：時的障礙的抽象概念會遭到移除，您的介面中有多少處於？ 當您知道使用者和其內容時，就會變更您的應用程式流程的哪些部分？ 功能表和按鈕的數目會維持？

例如，考慮共用體驗在混合實境等的 NASA 噴射推進實驗室內建的科學家 OnSight 工具中。 而不是建置系統，以查看火星 （抽象化拖曳到螢幕或顯示的資料） 的資料，他們可以享受有效地將它們放在 Mars 介面上，它們共同作業內的資料科學家。

![NASA OnSight](images/nasa-onsight.jpg)<br>
*NASA/JPL OnSight*

而不是尋找按鈕，以強調一些火星地質學，科學家可以指向它直接。 沒有功能表、 沒有畫筆，不具有效地使用工具的學習曲線。 利用我們已知的能力，從實體世界，在混合實境中的更自然互動可以規避深入的技術能力，甚至是最先進的業界工具中。

同樣地，語音和 AI 可以擴充自然的互動，就像這樣的經驗。 當數位助理可以 「 看到 」 到我們週遭世界時，交談會覺得較不麻煩。 Bot NASA 的工具中可能會將內容擷取科學家透過火星地質學，授予時科學家手勢和詢問有關 'this' 或 '，' 當做準備 （及交談式） 的資訊來源。 知道您的內容的電腦已準備好開始使用適用於您，透過最適合您的內容互動的方法資訊。

### <a name="building-on-a-foundation"></a>在基礎上建置

在這個世界情境體驗中，根據使用者的行動力是索引鍵。 流暢地切換裝置和服務，具有高度個人化的內容，使用者會周遊世界相關的實體順暢地，建立*大規模*平台的挑戰。 數位端點實體世界時，交互操作性統治擢。


> *「 結合數位 universe 和真實世界會解除鎖定人力可能...啟用每個個人和世界各地的組織，以實現更多功能。 」*<br>
> — Satya Nadella

Windows Mixed Reality 是為了建立裝置的生態系統的平台讓開發人員能夠建立沉浸式、 實惠且相容的體驗盡可能讓最多。 未來不會限制為單一的製造商，讓我們單獨進行單一裝置。 耳機，行動裝置、 電腦、 附屬應用程式... 所有這些實體的項目必須透過基礎平台以順利完成承諾的混合實境互通 （以及數位像是圖形和服務的詳細資訊）。

## <a name="areas-of-opportunity"></a>機會的區域

過去一年的開發人員合作關係已經發現的機會，resonate 與客戶，會建立成功企業解決方案的區域。 從科學家和設計工具和用戶端的技術人員，五個區域的機會問世以來，Microsoft 合作夥伴會與混合實境的尋找值。 這些區域已經提供 Windows Mixed Reality 等平台的未來需求的大規模深入，而且可以協助您了解這些新體驗將會如何影響方式我們了解、 共同作業、 溝通，並建立。

### <a name="1-creation-and-design"></a>1.建立和設計

其中一個主要的混合實境機會是能夠看到及管理 3D 設計即時項目，在真實世界設定中，則為 true 的大小和小數位數。 設計和原型設計工具有逸出畫面，將傳回給領域的設計通常保留給 wood 等 clay 的具體、 實體資料的範圍內。

<br>

>[!VIDEO https://www.youtube.com/embed/Hx6biWE2VsM]

[Autodesk](https://www.youtube.com/watch?v=Hx6biWE2VsM)建立目標在於改進共同作業，跨整個產品開發程序的混合的實境體驗。 適合工程師的能力和設計工具來調查並 critique 在其環境中的 3D 模型允許它們逐一查看即時的設計。 這不只啟用更快速建立原型，但在處理程序中提供給方法，以更有信心的決策。

體驗，例如此反白顯示的核心共同作業需要發生，使用者若要查看並與共用的物件通訊的能力。 Autodesk 的目標是要以數位的演出者、 學生和業餘愛好者展開工程師與專業的設計中的產品。 減少 3D 的專業知識的使用者層級，自然地與物件互動的能力會變得重要。

ThyssenKrupp 反白顯示的強大概念這裡重大、 內容相關的資訊可以快速地傳遞給使用者。 展望的體驗新的類別，來大量的可能是資訊的與使用者相關的內容將是資訊的索引鍵。
