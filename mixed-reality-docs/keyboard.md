# <a name="keyboard"></a>鍵盤

![鍵盤](images/UX/UX_Hero_Keyboard.jpg)

您的混合現實應用程式可以隨時叫用系統鍵盤。 請注意，系統鍵盤的行為會根據目標平臺的功能，例如，HoloLens 2 上的鍵盤支援直接手互動，而 HoloLens （第1代）上的鍵盤則支援注視和點按手勢輸入。
