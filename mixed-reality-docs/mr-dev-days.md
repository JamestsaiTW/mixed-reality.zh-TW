## <a name="about-altspace"></a>關於 Altspace

[AltspaceVR](https://altvr.com/) 是與來自世界各地的友好人們一起參加即時表演、聚會、酷炫課程的好地方。 這全都歸功於虛擬實境的神奇力量。  Altspace 在 2017 年成為 Microsoft 系列的一部分，我們非常興奮地展示如何在虛擬空間中進行活動。

Altspace 支援下列平台：HTC Vive；Oculus Rift、Go 和 Quest；Samsung GearVR；Windows Mixed Reality；以及電腦上的 2D 模式。  Mac 使用者必須使用 Parallels 或 Bootcamp，才能使用 Windows 存取 AltspaceVR。

### <a name="creating-an-account"></a>建立帳戶

建立帳戶的方式有兩種：

- 最有效率的方式是下載您的安裝程式的首選 AltspaceVR 用戶端、完成安裝，然後啟動用戶端。 進入此程式後，就會出現建立帳戶的選項。  您可以在[這裡](https://help.altvr.com/hc/en-us/articles/115003538414-Minimum-System-Specifications)找到最低系統規格。

- 另一個選項是造訪[線上帳戶註冊](https://account.altvr.com/users/sign_up)。

### <a name="troubleshooting"></a>疑難排解

如果您遇到任何 Altspace 相關問題，請查看[疑難排解頁面](https://help.altvr.com/hc/en-us/sections/115000792634-Troubleshooting)。 如果這無法解決您所遇到的問題，請提出[支援要求](https://help.altvr.com/hc/en-us/requests/new)。

### <a name="additional-tips"></a>其他提示

- 如果這是您第一次使用 AltspaceVR，請在第一個活動之前，挪出一些時間進行本教學課程。
- (選用) 我們建議您參加其中一個 AltspaceVR 101 活動，以取得更多秘訣與訣竅。
- 在活動之前，將感興趣的項目設為我的最愛/進行註冊，稍後可在用戶端輕鬆地找到。
- 與其他虛擬會議不同的是，AltspaceVR 是一種沉浸式環境，因此您可以封鎖其他出席者的觀看。 請尊重其他出席者的空間和觀看，以確保每個人都有良好的體驗。
- 將您想要在活動中交談的每個人設為朋友，如此一來，您便可再度邀請這些朋友回到您的 AltspaceVR 首頁，與您進行私人交談 (如果您想要的話)。
- 如果您站在麥克風附近，則您的聲音會廣播給活動中的所有人！ 因此，交談時請注意您所在的位置。

## <a name="frequently-asked-questions"></a>常見問題集
這一節會隨著我們向您收集到更多問題而經常更新！

**問** - 我可以在不使用 VR 頭戴式裝置的情況下參與嗎？  
**答** - 當然可以。 雖然我們覺得使用 VR 頭戴式裝置會有最佳的活動經驗，但我們很歡迎無法以該形式加入我們的人員參加。  Altspace 具備可用於加入活動的電腦用戶端。 ***Altspace 目前沒有可在 Mac 平台上運作的用戶端 (建議使用 Parallels 或 Bootcamp 來建立 Windows 安裝，以在 MAC 上使用 AltspaceVR***。  
  
**問** - 如何加入 2D 體驗？  
**答** - 在電腦上，您可選擇透過 Windows Store 或 Steam [下載 Altspace 用戶端](https://altvr.com/get-altspacevr/)。 下載用戶端之後，您需要建立帳戶並登入。 如需如何在 2D 體驗內導覽的資訊，請參閱[這裡](https://help.altvr.com/hc/en-us/articles/115003528373-How-do-I-control-my-avatar-with-Mouse-Keyboard-)。
  
**問** - 誰應該參加混合實境開發日  
**答** - 此活動歡迎每個在混合實境空間中工作或玩遊戲的人參加。 雖然大部分的內容都是以開發人員為主，但我們知道還有一些針對開發人員以外的各種角色所提供的有趣內容。 請記得檢查更新排程，以查看我們將要提供的內容種類。  

**問** - 我在 AltspaceVR 中找不到活動
