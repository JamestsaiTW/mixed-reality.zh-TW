# <a name="environment-considerations-for-hololens"></a>HoloLens 的環境考量

HoloLens 混合全像攝影版與 「 真實 」 世界中，置於您周圍環境的全像投影。 全像攝影版的應用程式視窗 「 擱置 」 在牆上、 全像攝影版 ballerina 旋轉手上、 兔子 어 位於您不知情的朋友標頭的頂端。 當您使用的擬真遊戲或應用程式時，在全像攝影版的世界會散佈到填滿周圍環境，但您仍然可以查看和空間四處移動。

您將全像投影會保持已放的位置，即使您關閉您的裝置。 

## <a name="setting-up-an-environment"></a>設定環境

HoloLens 裝置了解如何將穩定且精確全像投影所*追蹤*空間中的使用者。 但是不適當的追蹤，裝置不了解環境中或使用者，讓全像投影可以出現在錯誤的位置，不會出現在同一處就能每次，或根本未出現。

使用者所在的環境會經常會影響追蹤效能和調整環境，以引發穩定且一致的追蹤是封面，而不是一門學問。 許多不同的環境因素融合在一起以啟用追蹤，但身為 Mixed Reality 開發人員，有數個因素您可以保留在心，以調整空間，以利追蹤。
 
### <a name="lighting"></a>光源
Windows Mixed Reality 會使用 visual 的光線，來追蹤使用者的位置。 當環境是太亮，數位相機可以取得飽和，並不會出現。 如果環境是太深，相機無法取得足夠的資訊，並不會出現。 光源應該甚至和充分亮，人類看得見而不需要投入時間，但不是明亮光是查看痛苦。

區域有點亮的整體維度的區域也會有問題，因為觀景窗移動縮小和相應放大的亮空格時，調整。 這會導致裝置 「 取得遺失 」 和以為 light 中的變更視做等同於位置中的變更。 固定的 light 中的區域層級會導致更好的追蹤。

因為太陽可能隨時間變化相當大任何室外也可以在追蹤程式，導致不穩定。 比方說，在相同的空間，冬天和夏天的追蹤可以產生相當大的不同的結果，，像是在不同時間的年份，secondhand 燈光外的可能會更高。

如果您有 luxmeter，穩定的 500-1000 lux 會是不錯的起點。 

#### <a name="types-of-lighting"></a>類型的光源
不同類型的空間中的光線，也可以影響追蹤。 燈泡轉盤式與執行，如果 AC 頻率則 50 Hz，燈光脈衝在 50 Hz AC 電力。 人類看得，針對此閃爍是難以察覺。 HoloLens' 30fps 觀景窗會看到這些變更-一些畫面格會正確引發、 一些將會不佳亮起，但某些過度公開為相機嘗試補償光線跳動。

在美國，標準的電力頻率會是 60 赫茲，，因此 lightbulb pulse harmonized 與 HoloLens 的畫面播放速率，-60 赫茲 pulse 配合 Hololens' 30FPS 畫面播放速率。 不過，許多國家/地區有 AC 頻率的標準 50 Hz，表示要採取一些 Hololens 畫面格期間跳動，而有些則不會。 特別的是，在歐洲日光燈具有已知會造成問題。 

有幾件事，您可以嘗試解決情況的問題。 溫度、 球部年齡，以及準備循環是螢光燈閃爍的常見原因，並取代燈泡可以幫助。 也可協助強化燈泡，並確定目前的繪製為常數。 

### <a name="items-in-a-space"></a>在空間中的項目
HoloLens 也就是使用唯一的環境地標*功能*、 尋找本身空間中。 

裝置可以幾乎不會追蹤在不佳的功能區域中，裝置會有沒有辦法知道在空間中其所在。 將功能加入一個空格的背景牆，通常是很好的方法，來改善追蹤。 海報，符號久牆壁、 植物、 唯一的物件或其他類似的項目所有說明時。 凌亂的支援人員是很好的追蹤會導致環境的良好範例-有許多不同的功能，在單一區域中。 

此外，在相同的空間使用獨特的功能。 相同海報重複多次的牆，比方說，將會導致裝置混淆 HoloLens 也不會知道其中一個重複的海報查看。 新增的獨特功能的一種常見方式是空間的磁帶的使用遮罩，以建立唯一的行 nonrepetitve 牆壁和，以最低限度值的模式。 

如果您看到只要少量，請找您唯一到您自己的空間中，是場景的問問自己，一個好問題嗎？ 如果沒有的話，很可能在裝置將會一併追蹤的問題。

#### <a name="wormholes"></a>蟲孔
如果您有兩個區域或看起來相同的區域，追蹤程式可能會認為相同。 這會導致裝置本身矇騙以為它是在其他地方。 我們會呼叫這種重複的區域*蟲孔*。 

若要避免蟲孔，嘗試用來防止相同的區域，在相同的空間。 相同的區域有時會包括 factory 站台，建置、 伺服器機架或工作站上的 windows。 標籤區域，或將每個相似的區域中的獨特功能，可協助減輕蟲孔。
 
### <a name="temporal-stability-of-a-space"></a>時態性的穩定性的空格
如果您的環境是經常移位和變更，裝置就會有任何穩定的功能，可針對找出。 

中的空間，包括人員，就越容易遺失追蹤更多的移動物件。 移動傳送帶互連、 在不同狀態的建構，並有許多人在空間中的項目具有所有已知會造成追蹤問題。
 
### <a name="proximity-of-the-user-to-items-in-the-space"></a>鄰近程度使用者空間中的項目
同樣地將如何讓人無法也在接近眼睛的物件，HoloLens 的掙扎接近它的相機物件時。 如果物件是太接近可看到這兩種數位相機，或物件封鎖相機，裝置會出現更多的問題，並追蹤對物件。 

可以從物件的 15 cm 比不接近看到相機。
 
### <a name="surfaces-in-a-space"></a>在空間中的介面
強烈反光表面上看起來可能會影響追蹤的角度而有所不同。 將新的汽車-當您移動它，光會反映，而您會看到不同的物件，在介面中，當您移動時。 追蹤程式，以不同的物件反映介面代表變更的環境中，且裝置將會遺失追蹤。

較少閃亮的物件會針對所追蹤的工作變得更容易。

### <a name="wifi-fingerprint-considerations"></a>WiFi 指紋考量
只要啟用 WiFi 時，對應資料會與相互關聯 WiFi 指紋，即使未連線到實際 WiFi 的網路/路由器。 沒有 WiFi 資訊的空間和全像投影可能會稍微慢一點，來辨識。 如果 WiFi 訊號大幅變更，裝置可能會認為這完全是不同的空間。

網路識別 (也就是 SSID，MAC 位址) 不會傳送給 Microsoft，以及所有的參考會保留在 HoloLens 本機的 WiFi。

## <a name="mapping-new-spaces"></a>對應新的空間
當您輸入新的空間 （或載入現有的帳戶） 時，您會看到散佈到空間網狀圖形。 這表示您的裝置[對應鳥瞰](spatial-mapping-design.md)。 

如果您遇到放置全像投影的問題，請嘗試查核周圍的空間，因此 HoloLens 可以更全面地對應。 

如果您的 HoloLens 無法對應您的空間，或超出校正，您可以輸入 Limited 模式。 在 Limited 模式下，您將無法置於周圍環境的全像投影。

## <a name="environment-management"></a>環境管理
有兩個設定可讓使用者 「 清除 」 全像投影和 HoloLens 「 忘記 」 空間的原因。  它們存在於 「 全像投影和環境 」 中設定應用程式中，使用第二項設定也會出現在 [隱私權] 中設定應用程式。

1.  刪除全像投影附近 – 選取此設定，所有錨定全像投影和裝置位於其中的 「 目前的空間 」 的所有預存的對應資料，將會清除 HoloLens。  新的 [地圖] 區段會建立並儲存在該位置的資料庫，一旦全像投影會再次放在該相同的空間。

2.  刪除所有全像投影 – 選取此設定，HoloLens 會清除所有的對應資料並錨定全像投影整個資料庫中的空格。  將會重新探索任何全像投影和任何全像投影需要新放一次儲存在資料庫中的對應章節。

### <a name="managing-your-spaces"></a>管理您的空間

到單一資料庫中，儲存在本機上 HoloLens 裝置已摺疊的對應區段和不同的空間。 對應資料庫會安全地儲存，並僅適用於內部的系統，而不會的裝置，使用者即使插到電腦和/或使用檔案總管 應用程式的存取。 啟用 bitlocker 時，也會加密預存的對應資料。

全像投影會放在不同的位置，而不需要位置/全像投影之間的連線路徑時，就會存在多個對應元件。  全像投影錨定在相同的 [地圖] 區段中會被視為 「 鄰近 」 中目前的空間。

沒有開發人員若要匯出一小部分的 「 目前空間 」 的 API （目前辨識的對應元件的部分） 以啟用共用的全像案例。  目前沒有任何機制來下載整個資料庫的已對應的所有空格。


## <a name="hologram-quality"></a>全像品質

全像投影可以放在整個環境 — 高、 低和各地您 — 但是您會看到它們[全像攝影版的框架](holographic-frame.md)，位於前方眼睛。 若要取得最佳的檢視，請務必調整您的裝置，讓您可以看到整個畫面。 並盡速解決您的環境並探索 ！

為您[全像投影](hologram.md)尋找清晰、 清楚且穩定，您 HoloLens 靈敏度。 為您的需要。 當您第一次設定您的 HoloLens 時，將會引導完成此程序。 稍後如果全像投影看起來不正確，或您看到許多錯誤，您可以進行調整。

### <a name="calibration"></a>校正

如果您全像投影查詢畫面抖動或搖晃問題，或是如果您遇到放置全像投影的問題，請先嘗試[校正應用程式](calibration.md)。 如果您使用您的 HoloLens 時遇到任何 discomfort，也可協助此應用程式。

若要取得校正應用程式，前往 設定 > System > 公用程式。 選取 開啟校正，並遵循指示。

如果您執行校正應用程式，但仍發生全像品質問題，或您之所以看到頻繁的 < 追蹤遺失 」 訊息，請嘗試感應器調整應用程式。 移至 設定 > System > 公用程式，選取 開啟感應器調整，並遵循指示。

如果其他人會想要使用您的 HoloLens，應該執行校正應用程式先讓裝置已正確設定它們。
